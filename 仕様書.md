# 占い師ねずみBot (Fortune Mouse Bot) - プロダクト仕様書

## 1. プロダクト概要
「占い師ねずみBot」は、Discordコミュニティの活性化、ユーザーのウェルネス向上、およびサーバー内情報の最適化を目的とした多機能コミュニティマネジメントBotです。
愛らしいペルソナ（語尾「ちゅ」）をインターフェースとしつつ、バックエンドでは最先端の生成AI（Gemini API / ローカルLLM）と動的画像生成技術（Canvas / Satori）を組み合わせた高度な処理を行っています。

## 2. コア・テクノロジー
* **ハイブリッドAIシステム**: 
  * メインエンジンに `Gemini 2.5 Flash` を採用。APIエラーや制限発生時には、自動的にローカル環境のLLM (`Ollama` + `gemma2:9b` 等) にフォールバックする堅牢なエラーハンドリングを実装。
* **フルカスタム動的画像生成**: 
  * `@napi-rs/canvas` と `Satori` を併用し、AIの回答テキスト、ユーザーのステータス、ゲームの進行状況などをリアルタイムに計算し、1枚のリッチな画像（PNG/WebP）としてレンダリングして返信します。

## 3. 実装機能一覧（全モジュール詳細）

ユーザーは総合コマンド `/nezumi`（個人表示オプション `hidden` 対応）から、以下のすべての主要機能にアクセス可能です。

### 🔮 スピリチュアル解析モジュール（占い機能）
ユーザーのIDと日付をシード値とした疑似乱数と、AIによる自然言語生成を組み合わせたパーソナライズ占いを提供します。

* **タロットリーディング（1枚引き / 3枚引き）**
  * **1枚引き**: 全22枚の大アルカナから正位置・逆位置を判定。「カードの意味」「ねずみのささやき」「AIの特別解説」を1枚の画像にまとめて出力。
  * **3枚引き**: 過去・現在・未来の3枚を展開。スコア計算により運気の推移（V字回復、右肩上がり、警告など）を判定し、AIがストーリーを統合して解説。
* **星座占い**
  * 全12星座の毎日のランキング、ラッキーアイテム、スコアを算出。AIが全星座への一言コメントと「今日の抱負」を動的に生成し、ランキングボード画像として出力。
* **ルーン占い**
  * 全24種のルーン文字から1つをドロー（文字によって逆位置の有無を判定）。石に刻まれた意味とAIのお告げを、古代の石板風デザインの画像で出力。

### 🎮 エンターテインメント＆ユーティリティ
* **ヒット＆ブロー (数当てゲーム)**
  * モーダル入力を使用し、ランダムな4桁の数字を推理する論理ゲーム。Hit/Blowの判定を即座に返します。
* **一週間天気予報 (Open-Meteo API連携)**
  * ユーザーが入力した都道府県名から座標を割り出し、APIから1週間分の天気コード、最高/最低気温、降水確率を取得。ねずみ独自のコメントを添えた天気予報ボード画像を生成します。
* **アニマルギャラリー＆クイズ**
  * **写真表示**: マウス（ハツカネズミ等）、ラット（ドブネズミ等）、ねずみ以外の動物（カピバラ等）の3カテゴリからランダムに写真と名前を表示。
  * **ねずみクイズ**: ランダムに表示される動物の画像を見て「ねずみかどうか」をボタンで当てる2択クイズ。
* **ねずみ寿司**
  * **注文機能**: メニュー（マグロ、サーモン等）から注文すると、大将（ねずみ）の画像とともに、お寿司の写真と値段が載った提供画像を生成。
  * **おあいそクイズ（チキンレース）**: ランダムに設定された「目標金額」に対し、値段が隠された状態でお寿司を追加注文していき、ピタリ賞・ニアピン賞を狙うゲーム。

### ⚔️ コミュニティ育成型RPG「ネズミマスター」
JSONファイル（`pets.json`）をデータベースとして活用した、非同期型の育成・対戦システムです。

* **ゲット（捕獲）**: 5つの種族（カワウソ、ヒノネズミ、ミズネズミ、クサネズミ、エレキネズミ）からランダムに出現。50%の確率で捕獲可能。
* **ステータス**: HP、ATK、DEF、SPD、最大SP、混乱耐性（Stagger）、現在EXPをビジュアル化されたステータスカード画像として出力。
* **訓練（特訓）**: 5種類のコース（筋力、防御、走り込み、瞑想、総合）から選択。5分間のクールダウンタイムあり。経験値が貯まるとレベルアップし、コースに応じたステータスボーナスが付与されます。
* **バトル（PvP）**: 他のユーザーのペットと対戦。SPDに基づく行動順、ATK/DEFに基づくダメージ計算、SP消費による「必殺技」、Stagger値による混乱など、本格的なオートバトルが展開されます。
  * **リザルト生成**: 勝敗によりサーバー内のランキング（順位）が変動。AIによるバトルの「実況コメント」付きの勝敗結果画像を生成します。
* **ランキング**: サーバー内の上位10名のテイマーとペットのステータス一覧画像を生成。
* **お別れ（リリース）**: ペットを自然に還す（データを削除する）機能。専用のお別れ画像が生成され、ランキングが自動で繰り上がります。

### ☁️ ウェルネス・トラッキング「心の天気図」
ユーザーのメンタルヘルスを記録・可視化する機能です（`kibun.json`, `kibun_settings.json`）。

* **`/kibun` コマンド**: 現在の気分（レベル1〜5）と任意のテキストメモを記録。
* **`/kibun_setchannel` / `/resetchannel`**: 週次レポートの送信先（ユーザーに権限があるチャンネルのみリスト化）を設定・解除。
* **週次自動レポート (Cron)**: 毎週日曜日の22:00にバッチ処理が走り、1週間分の気分記録の推移と平均レベルをまとめ、ユーザーが指定したチャンネルへ自動送信します。

### 📢 動的インフォメーションボード「日替わり看板」
特定チャンネルの最下段に、常に最新のサーバー案内を固定表示する管理者向け機能です（`sticky.json`, `designs/`）。

* **自動追従システム**: メンバーが対象チャンネルに書き込むたびに、古い看板メッセージを削除し、最新の画像を一番下に再送信してピン留め状態を維持します。深夜0時の自動更新（Cron）にも対応。
* **`/event_setup` (モーダルUI登録)**: タイトル、日時、場所などをモーダルから入力し、3種類のSatoriテンプレート（チケット風、エレガント、ポップ）からデザインを選択して看板を予約登録。
* **`/design_upload` (高度なカスタム)**: 独自に作成したSatori互換のHTML/CSSファイルをアップロードし、特定の日付の看板として登録可能。
* **管理コマンド**: `/design_list` (登録リスト確認)、`/design_delete` (指定日のデザイン削除)、`/design_test` (本番前のプレビュー表示) を完備。Twemojiを用いたフルカラー絵文字の自動SVGレンダリングにも対応しています。

## 4. システム要件・使用技術
* **言語/環境**: Node.js
* **Discord API**: `discord.js` (v14)
* **AI連携**: `@google/generative-ai`, `axios` (Ollamaエンドポイント通信)
* **画像レンダリングパイプライン**: 
  * `@napi-rs/canvas` (複雑な文字組み、レイアウト計算)
  * `satori`, `satori-html`, `@resvg/resvg-js` (HTML/CSSからのSVG/PNG生成)
  * `sharp` (画像のリサイズ、WebP圧縮最適化)
* **スケジューラー**: `node-cron`
